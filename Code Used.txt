Matrix Math

To solve the problem of creating a new matrix that calculates the differences between immigration between country pairs, I used the following steps and code:

1. **Load the Matrix**: I loaded the matrix from the provided Excel file.
2. **Extract Countries**: I extracted the list of countries from the first column and first row of the matrix.
3. **Initialize a New DataFrame**: I created a new DataFrame to store the differences between immigration between country pairs.
4. **Calculate Differences**: I calculated the net number of immigrants moving from each origin country to each destination country and stored these values in the new DataFrame.
5. **Save the Differences Matrix**: I saved the new differences matrix to a new Excel file.

Here is the code I used:

```python
import pandas as pd

# Load the matrix from the Excel file
file_path = 'Matrix math.xlsx'
df = pd.read_excel(file_path, header=None)

# Extract the list of countries from the first column and first row
countries = df.iloc[1:, 0].tolist()

# Initialize a new DataFrame for the differences matrix
differences_matrix = pd.DataFrame(index=countries, columns=countries)

# Calculate the differences between immigration between country pairs
for i, origin in enumerate(countries):
    for j, destination in enumerate(countries):
        if i != j:
            # Calculate the net number of immigrants from origin to destination
            net_immigrants = df.iloc[i + 1, j + 1] - df.iloc[j + 1, i + 1]
            differences_matrix.loc[origin, destination] = net_immigrants
        else:
            differences_matrix.loc[origin, destination] = 0

# Save the differences matrix to a new Excel file
differences_matrix.to_excel('differences_matrix.xlsx')

print("Differences matrix saved to 'differences_matrix.xlsx'.")
```
